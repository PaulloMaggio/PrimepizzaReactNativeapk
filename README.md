# 🍕 Prime Pizza - App Mobile

Interface mobile oficial da PrimePizza, desenvolvida com **React Native** e **Expo**. O aplicativo permite que clientes naveguem pelo cardápio, visualizem categorias e façam pedidos de forma intuitiva.

## 🚀 Tecnologias Utilizadas

* **Framework:** React Native (Expo)
* **Linguagem:** TypeScript
* **Roteamento:** Expo Router (File-based routing)
* **Consumo de API:** Axios
* **Gerenciamento de Estado:** React Context API / Hooks
* **Estilização:** StyleSheet (padrão React Native)

## 📱 Funcionalidades

* **Cardápio Interativo:** Navegação fluida entre categorias (Pizzas, Bebidas, Sobremesas).
* **Consumo Dinâmico:** Integração direta com a API PrimePizza (Backend).
* **Experiência Nativa:** Otimizado para rodar em Android e iOS via Expo.
* **Componentização:** Estrutura modular para fácil manutenção e escalabilidade.
